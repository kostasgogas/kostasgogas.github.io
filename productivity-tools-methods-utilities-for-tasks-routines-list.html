Workflow Utilities

A List of Productivity Tools & Methods I use (or have used) for my work, my daily tasks & routines.

Enjoyed this List?

Buy me a Coffee

This is a list of software tools as well as method tools that I have used, and wanted to share with anyone needing some inspiration for something he doesn’t know how to do or haven’t thought existed.

Categories are divided by where I use them:

  • Everywhere
  • Advertising
  • Art
  • Design
  • Filmmaking
  • Website
  • Writing

Generic Tools

AutoHotkey

Free keyboard macro program. Supports hotkeys for keyboard, mouse, and joystick. Can expand abbreviations as you type them (AutoText).

Notes

Open Source.
  • Availability: Free
  • Platforms: Windows

Bulk Rename Utility

Free File Rename Utility for Windows. Rename multiple files with the click of a button. Batch renaming made easy.

  • Availability: 100% useable as free edition, You can pay too.
  • Platforms: Windows

YYYY-MM-DD

This is my way of naming files, because it helps sorting them out better by date.

Notes

This is a method, an approach, you can use it everywhere. + if you partner it up with AutoHotkey it makes a powerful combination.
  • Availability: Free
  • Platforms: All

TreeSize

Analyze your disk space.

Notes

I use this software because Windows 10 (in all the PCs I use never show me accurate information of how big or small (MB) files and folders are.
  • Availability: Free & Paid
  • Platforms: Windows

todo.txt

Track your tasks and projects in a plain text file.

Notes

This is a method, but there will be many apps that offer 'syntax-highlighting' for todo.txt.
  • Availability: Free
  • Platforms: All

On Screen Keyboard

A virtual keyboard, appearing on screen, letting you press the virtual keys with your mouse or touchscreen.

Notes

I use this when I my labtop is plugged in on a TV, away from my reach, so I don't have to get up.
  • Availability: Comes with Windows OS
  • Platforms: Windows (but surely other solutions exist)

Greenshot

A free screenshot tool optimized for productivity.

Notes

Open Source
  • Availability: Free & Paid
  • Platforms: Windows & MacOS

Veracrypt

Disk and file Encryption

Notes

Open Source.
  • Availability: Free
  • Platforms: Windows, macOS, Linux, freeBSD

Brave

Privacy focused browser for the the general public. Supports Tor.

Notes

Open Source.
  • Availability: Free
  • Platforms: macOS, Windows, Linux, Android, iOS

Protonmail

ProtonMail is the world's largest secure email service, developed by CERN and MIT scientists. Open source and protected by Swiss privacy law.

Notes

Open Source.
  • Availability: Free with 500MB storage and 150 messages per day, Donations Accepted, Paid Accounts from €4/Month
  • Platforms: Browser Based + Android & iOS App.

Standard Notes

A private notes app: simplicity, end-to-end encryption, extensions, and open-source applications.

Notes

Open Source
  • Availability: Free & Paid Features
  • Platforms: Web Based + Windows, macOS, Linux apps + Android & iOS apps.

WeblocOpener

Open MacOS Safari .webloc link-files on Windows and Debian GNU/Linux

Notes

I use this in Windows & Linux machines for opening internet shortcuts saved in Mac.
  • Availability: Free
  • Platforms: Windows, Linux

Markdown

A lightweight markup language with plain text formatting syntax.

Notes

It's a 'Syntax', a way of writing, a method, not a piece of software, it can't be paid and it can be written everywhere.
  • Availability: Free
  • Platforms: Everywhere

Fountain

Fountain is a plain text markup language for screenwriting.

Notes

It's a 'Syntax', a way of writing, a method, not a piece of software, it can't be paid and it can be written everywhere.
  • Availability: Free
  • Platforms: Everywhere

Sublime Text

A sophisticated text editor for code, markup and prose.

Notes

Proprietary
  • Availability: Free for evaluation, $80 per user (not machine) for continued use.
  • Platforms: Linux, MacOS, Windows

Afterwriting

Afterwriting provides post-processing tools for Fountain screenplays. (like converting Fountain format to PDF)

Notes

Open Source. There is also an offline version too. That's what I use.
  • Availability: Free
  • Platforms: Browser Based (I think everywhere, though haven't run [the offline version] on mobile)

Fountainhead

Contextual Sublime Text screenwriting environment using the Fountain screenplay syntax.

Notes

Open Source
  • Availability: Free
  • Platforms: Sublime text

SageThumbs

Thumbnail Preview for 160+ file formats (like .psd etc.) + Right click to convert image files too other formats.

  • Availability: Free
  • Platforms: Windows (10? maybe not)

Jekyll

Jekyll is a simple, blog-aware, static site generator for personal, project, or organization sites. Written in Ruby by Tom Preston-Werner, GitHub's co-founder, it is distributed under the open source MIT license.

Notes

Open Source
  • Availability: Free
  • Platforms: All

YAML

YAML is a human-readable data-serialization language.

Notes

Open Source
  • Availability: Free
  • Platforms: All

Markdown

Sublime Text 3 plugin for generating a Table of Contents (TOC) in a Markdown document.

Notes

Open Source
  • Availability: Free
  • Platforms: Sublime Text

Pocket

Pocket lets you save something you found on the web for later.

  • Availability: Free
  • Platforms: macOS, Windows, iOS, Android, Windows Phone, BlackBerry, Kobo eReaders, Browser.

Kaizen

The Japanese way of Continuous Improvement.

  • Availability:
  • Platforms: Life

LinkedIn Learning (ex lynda.com)

Video courses taught by industry experts in software, creative, and business skills

Notes

Really great content, have learned many many things through this excellent designed lessons.
  • Availability: Paid Membership
  • Platforms: Browser, iOS, Android

Broken Link Checker

Checks your websites and blogs for dead links

  • Availability: Free
  • Platforms: Browser

Moat

Analytics and measurement.

Notes

I use it mostly searching throught the Moat Ad Search to see ads ran by other companies.
  • Availability: Free & Paid
  • Platforms: Browser

Similar Web

Compare any website traffic statistics & analytics.

  • Availability: Free & Paid
  • Platforms: Browser

Google Analytics

Google Analytics lets you measure your advertising ROI as well as track your Flash, video, and social networking sites and applications

  • Availability: Free
  • Platforms: Browser, Android, iOS

HotJar

See how visitors are really using your website

Notes

I use it mostly for the 'Recordings' feature in order to identify User-Experience design weaknesses.
  • Availability: Free & Paid
  • Platforms: Browser

Data Merge

Merge data to create data-driven graphics through templates and variables in Illustrator

Notes

This is a really cool feature that lets you create designs and images from merging templates and csv files.
  • Availability: Paid (comes with Adobe Illustrator)
  • Platforms: Adobe Illustrator (and InDesign?)
